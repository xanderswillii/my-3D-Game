Knight's Isle
It is a 2D action-adventure game in which you play as a lone knight stranded on an uncharted island. Venture beyond your starting point to discover new lands, battle dangerous enemies, and collect treasures hidden throughout the world.
The game is currently in active development but can be played as a Demmo for the moment

Build with
Engine: Godot 4.6
Language: GDScript

Controls(QWERTY)
Move_left: A
Move_right: D
Move_up: W
Move_down: S

Devlopment Status
The game is currently in Development. Planned features to be added will be
- Additional enemy types
- More Islands to explore
- SFX and background music
- Treasures to find
